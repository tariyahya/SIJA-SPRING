# Dokumentasi Pendukung

Folder ini menyimpan artefak non-kode untuk membantu proses belajar:

- `ERD.md` – Deskripsi relasi tabel utama.
- `Flows.md` – Flow naratif untuk setup data, login, presensi RFID/QR/Face, dan laporan.
- `GitStrategy.md` – Panduan penggunaan GitHub bagi guru dan siswa.
- `Levels.md` – Daftar level pembelajaran dan indikator kelulusan.

Tambahkan gambar (ERD, wireframe, flowchart), file Postman, atau catatan tambahan sesuai kebutuhan kelas.
